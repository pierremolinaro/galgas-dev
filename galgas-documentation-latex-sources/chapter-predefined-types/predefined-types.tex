%--------------------------------------------------------------
\chapter{Predefined Types} \label{predefinedTypes}
%-------------------------------------------------------------

GALGAS predefines several types. This chapter presents all their features, including their constructors, readers, modifiers, methods, ...


\begin{description}
\item The predefined types are:
\begin{itemize}
\item \lienSectionType{@bool}, boolean objects;
\item \lienSectionType{@char}, Unicode characters;
\item \lienSectionType{@double}, floating point numbers;
\item \lienSectionType{@location}, whose value points out a location in a source file;
\item \lienSectionType{@sint}, the 32-bit signed integers;
\item \lienSectionType{@sint64}, the 64-bit signed integers;
\item \lienSectionType{@stringset}, set of \nomType{@string} objects;
\item \lienSectionType{@uint}, the 32-bit unsigned integers;
\item \lienSectionType{@uint64}, the 64-bit unsigned integers.
\end{itemize}
\end{description}
