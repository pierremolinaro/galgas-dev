\chapter{Types de base}\label{predefinedTypes}
%-------------------------------------------------------------

GALGAS predefines several types. This chapter presents all their features, including their constructors, readers, modifiers, methods, ...


\begin{description}
\item The predefined types are:
\begin{itemize}
\item \refTypePredefini{binaryset}, binary set objects (implemented with Binary Decision Diagrams);
\item \refTypePredefini{bool}, boolean objects;
\item \refTypePredefini{char}, Unicode characters;
\item \refTypePredefini{double}, floating point numbers;
\item \refTypePredefini{location}, whose value points out a location in a source file;
\item \refTypePredefini{object}, dont une instance peut encapsuler toute valeur ;
\item \refTypePredefini{sint}, the 32-bit signed integers;
\item \refTypePredefini{sint64}, the 64-bit signed integers;
\item \refTypePredefini{string}, the Unicode string objects;
\item \refTypePredefini{stringset}, set of \galgas{string} objects;
\item \refTypePredefini{type}, dont une instance repr√©sente un type ;
\item \refTypePredefini{uint}, the 32-bit unsigned integers;
\item \refTypePredefini{uint64}, the 64-bit unsigned integers.
\end{itemize}
\end{description}
