Simple install procedure for Unix :-----------------------------------% gzip -cd UNIX_TAR_FILE.tar.gz | tar xvf - # unpack the sources% cd GALGAS_PROJECT_DIR  # goto galgas directory% make all                     # build galgas and galgas_debug executables[ Become root if necessary]% make install                  # install galgas and galgas_debug executablesInstall Notes :---------------For Darwin (MacOS X), use cc instead of gcc : in the Makefile file,  changethe CC assignment "CC = gcc" to "CC = cc"."make install" copies galgas and galgas_debug executables into the '/usr/bin'directory ; for an other install directory, change the INSTALL_DIR variableassignment in the Makefile file.
