Note about compiling for Win32 target (CodeWarrior Pro 8) :-----------------------------------------------------------When you compile for Win32 target (Windows or Macintosh host), two compilation errors occur on winnt.h file :  Error   : data type is incomplete  winnt.h line 6861   EVENTLOGRECORD  pEventLogRecords[];  Error   : data type is incomplete  winnt.h line 6867   DWORD     ulOffsets[];           //the offsets from the start of this structure to the EVENTSFORLOGFILE structureJust change the two lines as following :   EVENTLOGRECORD  pEventLogRecords[1];   DWORD     ulOffsets[1];    And one compilation error occurs on rpcndr.h file :  Error   : data type is incomplete  rpcndr.h line 1195    unsigned char       Format[];Just change this line as following :    unsigned char       Format[1];
